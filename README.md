🧾 User Registration System (Python + Tkinter)

This is a simple Python GUI application that allows users to register by entering their Name, Email, and Password.
It stores the data securely in an SQLite database, ensuring each email is unique.
This beginner-friendly project helps practice GUI design, database handling, and error management using Python.

📊 Features

✅ User-friendly Tkinter interface
✅ Registers users with name, email, and password
✅ Stores data in an SQLite database
✅ Prevents duplicate email registration
✅ Shows success/error messages with messagebox
✅ Beginner-friendly and easy to understand

🔧 Technologies Used

Python

Tkinter (for GUI)

SQLite (for database)

📁 File

Register.py → Main Python script for the registration system

🚀 How to Run

Clone the repository:

git clone https://github.com/hibalodhids/tkinter-user-registration-system.git

cd tkinter-user-registration-system


Run the Python script:

python Register.py
